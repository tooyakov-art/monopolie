# Монополие

Командный центр бизнеса. Все сервисы, все аккаунты, все метрики — в одном месте.

## Структура

```
monopolie/
├── board.md          # Доска — все проекты/сервисы и их роль
├── accounts/         # Аккаунты по платформам
├── metrics/          # Метрики: контент, просмотры, доход
├── mini/             # Монополие-мини — виджет, только доходность
└── services/         # Каталог сервисов/инструментов
```

## Философия
- уСПЕХ = Успеть. Быстрее конкурентов
- Тестировать много ниш параллельно
- Каждый сервис = клетка на доске, играет свою роль
- Мини-виджет на телефон — доходность всегда перед глазами
