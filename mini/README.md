# Монополие-мини

Виджет на телефон. Только доходность. Всегда перед глазами.

## Что показывает
- Общий доход за месяц
- Доход по каждому аккаунту/проекту
- Тренд (растёт/падает)

## Реализация
PWA (Progressive Web App) — открывается как приложение на телефоне.
Тянет данные из metrics/. Минимальный UI, только цифры.
